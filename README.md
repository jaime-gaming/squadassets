An advanced, web-based asset browser for Squad Busters game files. This tool allows users to upload, index, and preview game assets (images, UI elements, and sprites) directly in the browser using a high-performance, responsive interface.

üöÄ How to Setup & Use

Follow these simple steps to get the explorer running on your computer:

Download the Project: Download this repository as a ZIP file (click the green "Code" button and select "Download ZIP").

Export/Extract: Extract the contents of the downloaded ZIP file to a folder on your computer.

Prepare the Folder: Ensure the index.html file and any other project files are in the same folder.

Run the App: Double-click the index.html file to open it in your web browser (Chrome, Edge, or Firefox recommended).

Load Assets: Once the page is open, use the sidebar to upload your Squad Busters Assets ZIP file to start browsing.
